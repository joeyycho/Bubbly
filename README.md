> **Make your macOS workspace cozy and playful.**

![Swift](https://img.shields.io/badge/Swift-5.9-orange?style=for-the-badge&logo=swift)

지루한 데스크탑 화면에 생기를 불어넣어 줄 맥 OS 전용 화면 꾸미기 앱, **버블리(Bubbly)**입니다. 공부나 업무 중에 잠시 멍하니 비눗방울을 터뜨리며 힐링하는 시간을 가져보세요.

---

## ✨ Features
*   **Floating Bubbles:** 앱 아이콘을 클릭할 때마다 화면 곳곳에 예쁜 비눗방울들이 생성됩니다.
*   **Satisfying Pop:** 마우스로 비눗방울을 클릭해 보세요. 기분 좋은 소리와 함께 방울이 터집니다.
*   **Instant Clear:** 화면이 너무 복잡해졌나요? 상단 메뉴바의 핀(Pin) 버튼으로 한 번에 정리하세요.
*   **Universal Build:** 인텔 맥(Intel)과 애플 실리콘(M1, M2, M3)을 모두 지원합니다.

---

## 🚀 Installation (설치 방법)
1.  [Releases](https://github.com/joeyycho/Bubbly/releases) 페이지에서 가장 최신 버전의 `Bubbly.zip`을 다운로드합니다.
2.  압축을 풀고 `Bubbly.app` 파일을 **응용 프로그램(Applications)** 폴더로 옮겨주세요.
3.  **처음 실행 시 주의사항 (Security Guide):**
    *   앱을 **우클릭 (Control + 클릭)** 한 뒤 **'열기(Open)'**를 선택하세요.
    *   보안 팝업창이 뜨면 다시 한번 **'열기(Open)'**를 눌러주세요.
    *   이 과정은 앱스토어 외부 배포 앱을 실행하기 위한 최초 1회 인증 절차입니다.
